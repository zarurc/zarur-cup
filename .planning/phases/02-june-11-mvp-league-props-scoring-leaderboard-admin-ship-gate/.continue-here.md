## Critical Anti-Patterns

| Severity | Pattern | Why |
|----------|---------|-----|
| **non-blocking** | npm install on Mac when CI is Linux can produce platform-incomplete lockfiles | Use `npm install --include=optional` (or regenerate in a Linux Docker container) to capture all platform-conditional deps |
| **non-blocking** | `supabase-js` realtime client requires native WebSocket; Node 20 lacks it | Pass `realtime.transport = ws` to `createClient()` in tests until CI moves to Node 22+ |
| **non-blocking** | GitHub Actions runners are IPv4-only тЖТ cannot reach Supabase direct connection (IPv6) | Always use the Supabase Session pooler URL in CI (`*.pooler.supabase.com:5432`) |
